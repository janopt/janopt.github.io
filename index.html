About the painting signature

The painting signature is the binary representation of the file
jano.pt_1_signature_hash.txt, which was created by the three following
steps.

Step 1 This file, jano.pt_1.txt, which describes the painting, was
digitally signed by jano.pt. The signature is in file
jano.pt_1_signature.txt.

Step 2 The SHA3-256 hash of file jano.pt_1_signature.txt was
obtained. This hash is in file jano.pt_1_signature_hash.bin.

Step 3 File jano.pt_1_signature_hash.bin was encoded with base64 URL
safe encoding. The result is in file jano.pt_1_signature_hash.txt.

Four one bits were added to the corners of each binary
representation. These extra one bits are not part of the binary
representations.

The one bits, that were removed from this painting, will be glued to
the next painting (except the bottom right corner that jano.pt kept
for himself).

The content of the signature file jano.pt_1_signature.txt will be
included in the next painting description file.

A chain of paintings is created! Both physically, through each
painting's signature, and digitally, through each description file's
signature. :-)

Sobre a assinatura do quadro

A assinatura do quadro é a representação binária do ficheiro
jano.pt_1_signature_hash.txt, que foi criado pelos três passos que se
seguem.

Passo 1 Este ficheiro, jano.pt_1.txt, que descreve o quadro, foi
assinado digitalmente pelo jano.pt. A assinatura está no ficheiro
jano.pt_1_signature.txt.

Passo 2 Foi obtido o código hash SHA3-256 do ficheiro
jano.pt_1_signature.txt. Este código hash está no ficheiro
jano.pt_1_signature_hash.bin.

Passo 3 O ficheiro jano.pt_1_signature_hash.bin foi codificado com a
codificação base64 URL safe. O resultado está no ficheiro
jano.pt_1_signature_hash.txt.

Foram acrescentados quatro bits um aos cantos de cada representação
binária. Estes bits um extra não fazem parte das representações
binárias.

Os bits um, que foram retirados deste quadro, vão ser colados no
próximo quadro (exceto o canto inferior direito que o jano.pt guardou
para si).

O conteúdo do ficheiro jano.pt_1_signature.txt vai ser incluído no
ficheiro com a descrição do próximo quadro.

Fica criada uma cadeia de quadros! Tanto fisicamente, através da
assinatura de cada quadro, como digitalmente, através da assinatura de
cada ficheiro de descrição. :-)
